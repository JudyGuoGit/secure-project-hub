🎯 WHAT THIS TESTS
──────────────────

PKI/mTLS Authentication - Certificate-based security where:
  • Client proves identity with certificate (not password)
  • Server verifies certificate against trusted CA
  • Mutual TLS - both sides authenticate
  • No tokens needed - certificate IS the credential
  • Suitable for service-to-service communication


💼 PRODUCTION NOTES
────────────────────

For production:

  • Implement certificate rotation
  • Set up monitoring for expiration
  • Use certificate revocation lists (CRL)
